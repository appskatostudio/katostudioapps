Política de Privacidade

Jogue à vontade.
Sem rastreamento.

O Memory Fast Game não manda seus dados pra lugar nenhum. Recordes ficam no celular, só seus.

Última atualização: maio de 2026  ·  Memory Fast Game v1.0

Resumo

O Memory Fast Game não coleta dados pessoais. Seu progresso é salvo apenas no dispositivo e nenhuma informação é enviada a servidores externos.

01

Introdução

Memory Fast Game é um jogo de memória para Android desenvolvido por Gustavo Kato. Esta política descreve como o aplicativo trata as informações do usuário.

Nenhum dado pessoal é coletado, transmitido ou armazenado fora do dispositivo.

02

O que o app não coleta

  • Não exige cadastro, login ou conta de usuário
  • Não coleta nome, e-mail ou qualquer dado de identificação pessoal
  • Não rastreia localização, contatos ou histórico de navegação
  • Não acessa câmera, microfone ou arquivos do dispositivo
03

Dados armazenados localmente

O jogo salva as seguintes informações exclusivamente no seu celular:

  • Recordes e pontuações: melhor tempo e número de jogadas por nível
  • Progresso: níveis desbloqueados e configurações de dificuldade
  • Preferências: configurações de som e vibração

Para apagar tudo: Configurações → Aplicativos → Memory Fast Game → Armazenamento → Limpar dados.

04

Anúncios — Google AdMob

O app exibe anúncios via Google AdMob. O AdMob pode coletar dados de identificação do dispositivo para mostrar anúncios relevantes, conforme a Política de Privacidade do Google.

O Memory Fast Game não tem acesso a esses dados e não os repassa a terceiros.

Para desativar anúncios personalizados: Configurações do Android → Google → Anúncios → Desativar personalização.

05

Permissões utilizadas

  • INTERNET: exclusivamente para carregar os anúncios do AdMob.
  • VIBRATE: feedback tátil ao virar as cartas, quando ativado pelo usuário.
06

Menores de idade

O Memory Fast Game pode ser jogado por crianças. Não coletamos intencionalmente dados de menores de 13 anos. Os anúncios do AdMob seguem as diretrizes de conteúdo voltado a crianças do Google Play.

07

Alterações nesta política

Qualquer atualização será publicada aqui com a data revisada. Vale a pena checar de vez em quando.
