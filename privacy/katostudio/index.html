Política de
Privacidade.

Este site não coleta dados pessoais, não cria contas e não rastreia usuários. O que acontece aqui, fica aqui.

Última atualização: maio de 2026 · katostudioapps
Resumo: este site é um portfólio estático. Não coletamos dados de navegação, não usamos cookies de rastreamento e não armazenamos nenhuma informação pessoal. Contribuições financeiras são processadas pelo Stripe — nós não vemos os dados do seu cartão.
01

O que é este site

O KatoStudio Apps (appskatostudio.github.io/katostudioapps) é o portfólio e a página de apoio do desenvolvedor independente Gustavo Kato. O site apresenta os aplicativos Android publicados e oferece uma forma de contribuição voluntária via Stripe.

02

Dados que não coletamos

  • Não criamos conta ou perfil de usuário
  • Não coletamos nome, e-mail ou qualquer dado de identificação
  • Não usamos cookies de rastreamento ou publicidade
  • Não instalamos scripts de analytics (Google Analytics, Hotjar etc.)
  • Não rastreamos o comportamento de navegação entre páginas
03

Hospedagem — GitHub Pages

Este site é hospedado no GitHub Pages. O GitHub pode registrar dados básicos de acesso (endereço IP, navegador, referrer) em seus servidores conforme a Política de Privacidade do GitHub. O KatoStudio não tem acesso a esses logs.

04

Contribuições — Stripe

A página de apoio (coffee/) redireciona para o Stripe ao escolher uma contribuição. Todo o processamento de pagamento ocorre nos servidores do Stripe, certificado PCI-DSS nível 1.

O KatoStudio não armazena dados de cartão, não tem acesso ao número completo e não processa pagamentos diretamente. Após a conclusão, recebemos apenas uma confirmação de valor e data via Stripe. Consulte a Política de Privacidade do Stripe para detalhes.

05

06

Menores de idade

Este site não é direcionado a menores de 18 anos. Contribuições financeiras devem ser realizadas por adultos ou com autorização dos responsáveis.

07

Alterações nesta política

Atualizações serão publicadas nesta página com a data de revisão atualizada.
